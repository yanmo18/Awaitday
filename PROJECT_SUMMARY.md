# Awaitday - 精美倒计时小程序

> 记录每一个重要时刻，用最美的方式等待

## 项目概述

**Awaitday** 是一款专注于视觉体验的倒计时小程序，支持多种展示样式和自定义配色。无论是生日、假期、出行还是任何重要日子，都能用最优雅的方式呈现。

### 核心特性

- **4种倒计时样式**：圆环进度、翻牌动画、渐变卡片、极简风格
- **18种预设颜色**：支持自定义主色调和渐变色
- **翻牌动画**：CSS 3D transform 实现的翻页时钟效果
- **圆环进度**：SVG 绘制的环形进度条
- **本地存储**：数据持久化，自动保存/加载

## 功能详解

### 1. 首页（倒计时列表）

- 显示所有已创建的倒计时
- 实时更新剩余时间（天/时/分/秒）
- 根据全局样式设置渲染对应组件
- 支持删除倒计时
- 圆形添加按钮

### 2. 添加页

- 输入倒计时名称
- 选择目标日期（年月日）
- 选择颜色（18色调色盘）
- 支持自定义主色调和渐变色
- 实时预览效果

### 3. 样式页

- 4种展示样式可选
- 点击即可切换，全局生效
- 样式预览效果展示

## 样式组件说明

### 1. 圆环进度 (RingStyle)

```
┌─────────────────┐
│   生日倒计时    │
│    ◐ 07 天     │  ← SVG圆环 + 中心数字
│   12:30:45     │
└─────────────────┘
```

- SVG 绘制圆环
- 渐变色进度条
- 中心显示天数
- 底部显示时分秒

### 2. 翻牌动画 (FlipCardStyle)

```
┌─────────────────┐
│   生日倒计时    │
│  ┌──┐┌──┐ 天   │  ← 数字卡片 + 右侧单位
│  │07│    │     │
│  └──┘└──┘      │
│   12:30:45     │
└─────────────────┘
```

- 数字带主题色背景
- "天"字灰色无背景
- 圆角卡片设计
- 时分秒底部显示

### 3. 渐变卡片 (GradientStyle)

```
┌─────────────────┐
│   生日倒计时    │
│ ═══════════════ │  ← 渐变背景
│      07         │  ← 大数字
│     天          │
└─────────────────┘
```

- 渐变背景大卡片
- 超大数字显示
- 视觉冲击力强

### 4. 极简风格 (MinimalStyle)

```
┌─────────────────┐
│   生日倒计时    │
│                 │
│      07 天      │  ← 纯数字
│                 │
└─────────────────┘
```

- 简洁数字展示
- 无多余装饰
- 干净利落

## 数据结构

### Countdown 倒计时数据

```typescript
interface Countdown {
  id: string           // 唯一标识
  name: string         // 倒计时名称
  targetDate: string   // 目标日期 (YYYY-MM-DD)
  theme: string        // 预设主题 ID
  primaryColor: string // 主色调
  secondaryColor: string // 渐变色
  bgColor: string      // 背景色
  style?: string       // 样式 (ring/flip/gradient/minimal)
  createdAt: string    // 创建时间
}
```

### 存储方式

- 使用 `Taro.getStorage` / `Taro.setStorage` 持久化
- 倒计时数据：`countdown_data`
- 样式设置：`countdown_style`
